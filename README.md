# Sketch & Sus

Multiplayer drawing bluff game inspired by the supplied `script.docx`, logo, and palette assets.

## Run

```bash
npm start
```

Then open `http://localhost:3000`.

## Built In

- Native WebSocket multiplayer rooms with shareable room links
- Inkster, Chameleon, and Catcher role flow
- Server-side timers, scoring, vote lock, and final Catcher judgement
- Weighted reservoir role sampling with Box-Muller Gaussian noise
- Canvas drawing, undo, clear, blur feed for Chameleon, and live Catcher wall
- Browser-native SFX with a mute toggle
